# AppUpdater

[![](https://jitpack.io/v/journe/AndroidAppUpdater.svg)](https://jitpack.io/#journe/AndroidAppUpdater)

封装了APP检查更新的相关逻辑

## 使用方法

**Step 1.** 添加 JitPack repository

```groovy
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

**Step 2.** 添加依赖

```groovy
	dependencies {
	        implementation 'com.github.journe:AndroidAppUpdater:0.1.0'
	}
```
